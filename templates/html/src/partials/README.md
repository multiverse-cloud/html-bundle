# Partials

Reusable TailAdmin-style partials live here as source documentation for the static bundle.

- `header/`: top navigation, search, notifications, dark mode, profile actions.
- `sidebar/`: dashboard navigation, groups, submenus, responsive mobile drawer.

The production bundle currently injects these partials from `common-header.js` and `common-sidebar.js` to keep all 116 pages DRY.
